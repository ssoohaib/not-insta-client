# not-insta-client


## Installation

Install not-insta-client with npm

```bash
  git clone https://github.com/ssoohaib/not-insta-client.git
  cd ./not-insta-client
  git checkout master
  npm install
```

## Run


```bash
  npx expo start
```

## .env

Server public url,
```bash
EXPO_PUBLIC_BASE_URL=
```
